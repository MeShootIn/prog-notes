ctrl + shift + f - поиск в Windows Terminal
alt + shift + = — сплит панели с профилем по умолчанию справа от текущей
alt + shift + - — сплит панели с профилем по умолчанию снизу от текущей
alt + shift + стрелки — изменение размера панели
alt + стрелки — перемещение между ПАНЕЛЯМИ (не вкладками)
Ctrl + w — закрыть панель
Ctrl + f - поиск
