### ПОИСК ###


* Исключение из поиска:
замок купить -англия
htmlspecialchars -site:qaru.site


ДАЛЕЕ "-" - ПРОБЕЛ

* Поиск только по тексту, который лежит между тегами <title>.
intitle:выращивание-сайтов

* По содержимому url (с точностью до url-символов):
inurl:Преобразование_Хафа

* Поиск только по тексту, который лежит между тегами <body>.
intext:создание-сайтов-для-предпринимателей

* Показывает последнюю копию вебстраницы (из кэша Google), которую проиндексировал Google.
cache:http://mosvpiska.navalny.ru/

* Поиск ведётся по типу файлов и заданного поиского значения.
Hough transform filetype:pdf

* Показывает похожие страницы.
related:youtube.com

* Поиск на определённом сайте.
кекс site:youtube.com

* Ищет определение.
define: преобразование хафа


### CHROME ###


Ctrl + T - открыть новую вкладку
F6 - перейти к адресной строке (лучше, чем закрывать текущую и создавать новую вкладку для нового запроса)
Ctrl + Tab - некст вкладка
Ctrl + W - закрыть вкладку
Shift + Delete - удалить подсказку
Ctrl + O - открыть файл с компа (pdf, doc[x], ...)
F11 - фуллскрин
Ctrl + ЛКМ - открыть ссылку в новой вкладке
